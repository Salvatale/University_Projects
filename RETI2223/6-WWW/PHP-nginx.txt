Installiamo il seguente package: php-fpm
Comando: sudo apt install php-fpm

Ho modificato il file /etc/nginx/sites-available/default con:

	location ~ .php$ {
		include snippets/fastcgi-php.conf;
		fastcgi_pass unix:/var/run/php/php-fpm.sock;
	}

Ho scaricato tutti i file in http://didattica-linux.unipr.it/~roberto.alfieri@unipr.it/matdid/RETI/http/php/ nella directory /var/www/vh1 con wget.
